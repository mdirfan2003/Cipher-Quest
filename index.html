Training Manual

How to Play

Read the encrypted message, work out the original text, and submit your answer before time runs out.

Caesar Cipher

Each letter is shifted forward or backward by a fixed number of places in the alphabet.

Vigenere Cipher

A keyword controls how far each letter is shifted, making the message harder to decode.

Binary to Text

Letters are shown as groups of binary numbers. Each group can be converted back into a character.

Morse Code

Letters are represented using dots and dashes. Spaces separate letters and words.

ROT13

Each letter is shifted by 13 places. Applying ROT13 again returns the original message.

Atbash

The alphabet is reversed, so A becomes Z, B becomes Y, and C becomes X.
